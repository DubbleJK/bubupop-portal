# 네이버 SEO 블로그 글 생성기 — 실행 방법 (처음 쓰시는 분용)

코딩을 해본 적이 없어도 아래 순서대로만 하시면 됩니다.

---

## 1단계: 폴더 열기

1. **파일 탐색기**를 엽니다 (Windows 키 + E).
2. **C:\Users\USER\bubucalculate-app** 폴더로 이동합니다.
3. 이 폴더 안에 있어야 합니다. (quotation-app이 아니라 **bubucalculate-app**)

---

## 2단계: 터미널(검은 창) 열기

1. **bubucalculate-app** 폴더 창에서 **주소창**을 한 번 클릭합니다.
2. 주소창에 `cmd` 라고 입력한 뒤 **Enter** 키를 누릅니다.
3. 검은색 창(명령 프롬프트)이 열립니다.

또는:

- **bubucalculate-app** 폴더를 연 상태에서 **Shift + 오른쪽 클릭** → **"여기에 PowerShell 창 열기"** 또는 **"여기에 명령 프롬프트 열기"**를 선택해도 됩니다.

---

## 3단계: 패키지 설치 (처음 한 번만)

검은 창에 아래 문장을 **그대로** 입력한 뒤 **Enter** 키를 누릅니다.

```
npm install
```

- 영어와 띄어쓰기를 정확히 입력해야 합니다.
- 끝날 때까지 기다립니다 (1~2분 걸릴 수 있습니다).
- "added ... packages" 같은 메시지가 나오면 성공입니다.

---

## 3-2단계: OpenAI API 키 설정 (블로그 글 생성용)

블로그 글은 **OpenAI(ChatGPT)** 로 생성됩니다. API 키가 있어야 합니다.

1. **키 발급**: 브라우저에서 [https://platform.openai.com/api-keys](https://platform.openai.com/api-keys) 에 접속해서 로그인한 뒤 **Create new secret key** 로 키를 만듭니다. (유료 결제 수단 등록이 필요할 수 있습니다.)
2. **키 저장**:
   - **bubucalculate-app** 폴더 안에 **.env.local** 이라는 파일을 만듭니다. (메모장으로 만들어도 됩니다.)
   - 그 안에 아래 한 줄만 넣고, `sk-...` 부분을 본인 키로 바꿔 넣습니다.
   ```
   OPENAI_API_KEY=sk-여기에_발급받은_키_붙여넣기
   ```
   - 저장한 뒤, **프로그램을 다시 실행**할 때(4단계 `npm run dev`) 이 키가 적용됩니다.
3. **주의**: .env.local 파일은 다른 사람에게 보여 주지 마세요. 키가 노출되면 요금이 부과될 수 있습니다.

---

## 4단계: 프로그램 실행

같은 검은 창에 아래를 입력하고 **Enter** 키를 누릅니다.

```
npm run dev
```

- "Ready in ..." 또는 "Local: http://localhost:3001" 이 보이면 **실행된 것**입니다.
- **이 검은 창은 닫지 마세요.** 닫으면 프로그램이 꺼집니다.

---

## 5단계: 브라우저에서 열기

1. **인터넷 브라우저**(크롬, 엣지 등)를 엽니다.
2. 주소창에 아래를 입력합니다.

   ```
   http://localhost:3001
   ```

3. **Enter** 키를 누릅니다.
4. **부부계산 포털** 화면이 보이면 성공입니다.

---

## 6단계: 블로그 글 생성기 쓰기

1. 포털 화면에서 **"네이버 SEO 블로그 작성"** 카드를 클릭합니다.
2. **블로그 글 생성** 페이지로 이동합니다.
3. 다음을 입력합니다.
   - **메인 키워드**: 예) 스티커 제작
   - **서브 키워드**: (원하면) 3개까지
   - **업종 템플릿**: 스티커 / 배너 / DTF / UV / 의류 등 중 선택
   - **강점**: 당일제작, 소량맞춤, 오랜경험 등 원하는 것 체크
   - **지역**: 예) 부산
   - **타겟**: 예) 소상공인, 급한 고객
   - **글 길이**: 1000자 / 1500자 / 2000자 중 선택
   - **말투**: 친근 / 전문 / 후기 / 가이드 중 선택
4. **"블로그 글 생성하기"** 버튼을 누릅니다. (OpenAI로 생성하므로 몇 초~십 초 걸릴 수 있습니다.)
5. 아래가 나옵니다.
   - **제목 후보 5개** → 마음에 드는 것 복사해서 사용
   - **본문** → "📸 여기에 사진" 자리에 실제 사진을 넣으면 됨. 전체 복사 후 네이버 블로그에 붙여넣기
   - **해시태그 10개** → 복사해서 블로그 글 하단에 붙이기

## 자주 묻는 질문

**Q. 검은 창을 실수로 닫았어요.**  
→ 다시 **2단계~4단계**를 해서 `npm run dev` 를 실행하면 됩니다.

**Q. "OPENAI_API_KEY가 설정되지 않았습니다" 라고 나와요.**  
→ **3-2단계**를 진행해 주세요. **bubucalculate-app** 폴더 안에 **.env.local** 파일을 만들고, 그 안에 `OPENAI_API_KEY=발급받은키` 한 줄을 넣은 뒤, **검은 창을 닫았다가 다시 `npm run dev`** 로 실행하세요.

**Q. "npm을 찾을 수 없습니다" 라고 나와요.**  
→ **Node.js**가 설치되어 있지 않은 것입니다.  
   [https://nodejs.org](https://nodejs.org) 에서 **LTS** 버전을 다운로드해 설치한 뒤, 컴퓨터를 한 번 재시작하고 다시 **3단계**부터 진행하세요.

**Q. 계산기(견적)는 어디서 켜요?**  
→ 포털 첫 화면의 **"견적 계산기"** 카드를 누르면 견적 앱이 새 탭에서 열립니다.  
   (견적 앱은 **quotation-app** 폴더에서 따로 `npm run dev` 로 켜 두어야 할 수 있습니다.)

**Q. 만든 글이 내 블로그 스타일과 다르면?**  
→ 지금 버전은 "당일 제작", "소량 맞춤", "신뢰", "10년 경력" 등을 강조하는 문장으로 맞춰져 있습니다.  
   블로그에 올리신 뒤 문장만 조금 고치시면 됩니다.
